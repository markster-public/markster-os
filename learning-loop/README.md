# Learning Loop

The learning loop turns raw business material into approved business knowledge.

## Core Principle

Raw material is not canon.

The flow is:

1. capture raw material
2. extract candidate updates
3. review candidates
4. promote approved updates into canon
5. let skills read canon only

## Folders

- `inbox/`: raw material, never canonical
- `candidates/`: structured proposed updates
- `canon/`: approved knowledge only
- `prompts/`: extraction and promotion prompts
- `promotion-rules.md`: promotion gate
