# Stage 2: Growing ($100K-$500K) -- Implementation Guide

> From ScaleOS Operational Handbook v1.

---

## Situation

Revenue is happening. Seeking consistency. Team forming. Founder still does most things but systems are starting.

---

## Priority Focus

1. **Full Growth pillar systematized** (G1 + G2 + G3 working as integrated pipeline)
2. **O2: Automate** repetitive work (save 20+ hours/week)
3. **O3: Instrument** -- build metrics dashboard with North Star metrics

---

## Timeline

| Month | Focus | Outcome |
|-------|-------|---------|
| Month 1 | Systematize Find + Book (automated pipeline generation) | Qualified prospects flowing weekly without founder manual work |
| Month 2 | Build first automations (save 10+ hrs/week) | Content pipeline automated. Outreach sequences running. |
| Month 3 | Create dashboard with North Star metrics | Weekly scorecard visible. Know what's working. |

---

## Success Signal

**20+ hours/week recovered through systems.**

---

## Common Mistake

**Hiring before systemizing (multiplies chaos).**

In AI-native context: building more agent capabilities before the existing ones produce consistent results.

---

## What "Done" Looks Like for Each Brick at Stage 2

| Brick | Done When... |
|-------|-------------|
| G1: Find | 50+ qualified prospects/week flow in without founder building lists manually |
| G2: Warm | Content publishes on cadence (LinkedIn 5x/week, blog 2x/month). Nurture sequences running. |
| G3: Book | Outreach sequences running. 5+ meetings/month from combined inbound + outbound. |
| O2: Automate | Content generation, outreach, and reporting automated. 20+ hrs/week saved. |
| O3: Instrument | Dashboard showing: meetings booked, pipeline value, content performance, MRR. Weekly review happening. |
